== Installation ==

1. Upload `notify` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Turn on plugin 
4. Set Subject & Body of message
5. Set fadeIn & fadeOut delay of notification
